========================
Project 01: it.architect
========================

Year: 2014
Type: Webapp, frontend development, interface design, mobile
Languages: HTML, CSS/LESS, Java EE, Javascript/jQuery
Tools: Adobe Photoshop, Netbeans, Coda, CodeKit

Description:

This was a university project for a programming class I worked on in 2014. I was part of a team of ten students who had to develop a webapp for itelligence called 'it.architect'.

This webapp was created from scratch in Java EE; I was the leader of our interface/design sub-team and responsible for developing the interface and frontend code.
